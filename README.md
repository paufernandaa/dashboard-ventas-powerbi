# 📊 Dashboard de Ventas en Power BI

## Descripción
Desarrollé un dashboard en Power BI para analizar datos de ventas y detectar patrones relevantes para la toma de decisiones.

## Objetivo
Visualizar el comportamiento de las ventas por categorías, regiones y períodos de tiempo.

## Desafío
Transformar datos en bruto en información clara y fácil de interpretar.

## Solución
Se creó un dashboard interactivo que permite analizar tendencias, comparar categorías y evaluar el rendimiento de ventas.

## Herramientas
- Power BI  

## Aprendizajes
- Visualización de datos  
- Análisis de tendencias  
- Creación de dashboards interactivos  

## Métricas de impacto
El dashboard permitió identificar patrones de ventas y destacar categorías con mayor rendimiento, facilitando el análisis de la información en menor tiempo.

## Habilidades aplicadas
- Análisis de datos  
- Visualización de datos  
- Pensamiento analítico  
- Uso de herramientas de BI  

## Justificación del proyecto
Elegí este proyecto porque representa mi crecimiento en el análisis y visualización de datos, integrando herramientas utilizadas en el ámbito profesional y demostrando mi capacidad para transformar datos en información útil.

## Conclusión
El proyecto permitió identificar patrones de ventas y facilitar la toma de decisiones basada en datos.
